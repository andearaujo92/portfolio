Projects and studies

Heart failure prediction

In this project I worked with Logistic regression to predict a binomial response about Heart failure prediction. I also created some questions to answer through visualizations.

Revenue prediction - Time series

I worked in this project in order to get a revenue prediction through time series techniques. It was compared two models, Auto regressive model and ARIMA model to see results and visualizations.

Laptop price prediction

This project needed a good work in Featuring engineering to get all the features about the laptops. I created some questions to be answered through visualizations. The data was preprocessing through sklearn library and predicted with Xgboost model.

New market, new clustering

In this project I helped the manager to get a new cluster for potential customers in order to understand the market. I worked with Preprocessing techniques. It was applied elbow technique in order to get the better cluster and Kmeans model to predict the clustering. After predicting, it was created a grid chart to understand each customer cluster.

Titanic deads prediction

This is a classical competition on Kaggle where I worked applying EDA, One hot Encoder, Simple imputer and Pipelines. For prediction it was applied XGB Classifier and it got a good accuracy.

Pizza price prediction

In this project I worked on a pizza prices dataset from Kaggle, answering some questions and creating a Random Forest model in order to predict price and understand the effects of each feature
